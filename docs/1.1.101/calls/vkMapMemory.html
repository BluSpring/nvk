 Categories vkMapMemory Map a memory object into application address space Syntax 
VkResult vkMapMemory();
Parameters device VkDevice is the logical device that owns the memory. memory VkDeviceMemory is the VkDeviceMemory object to be mapped. offset Number is a zero-based byte offset from the beginning of the memory object. size Number is the size of the memory range to map, or VK_WHOLE_SIZE to map from offset to the end of the allocation. flags Number is reserved for future use. ppData Object.$(BigInt) is a reference in which is returned a host-accessible reference to the beginning of the mapped range. This reference minus offset must be aligned to at least VkPhysicalDeviceLimits::minMemoryMapAlignment.
