Properties VkExternalFenceProperties.prototype.sType Number VkExternalFenceProperties.prototype.pNext null VkExternalFenceProperties.prototype.exportFromImportedHandleTypes Number is a bitmask of VkExternalFenceHandleTypeFlagBits indicating which types of imported handle handleType can be exported from. VkExternalFenceProperties.prototype.compatibleHandleTypes Number is a bitmask of VkExternalFenceHandleTypeFlagBits specifying handle types which can be specified at the same time as handleType when creating a fence. VkExternalFenceProperties.prototype.externalFenceFeatures Number is a bitmask of VkExternalFenceFeatureFlagBits indicating the features of handleType.
