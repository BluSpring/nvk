Parameters object When instantiated with an object, members can be filled directly. Properties VkMemoryRequirements.prototype.size Number is the size, in bytes, of the memory allocation required for the resource. VkMemoryRequirements.prototype.alignment Number is the alignment, in bytes, of the offset within the allocation required for the resource. VkMemoryRequirements.prototype.memoryTypeBits Number is a bitmask and contains one bit set for every supported memory type for the resource. Bit `i` is set if and only if the memory type `i` in the VkPhysicalDeviceMemoryProperties structure for the physical device is supported for the resource. Default Properties VkMemoryRequirements.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkMemoryRequirements.memoryLayout Object Object describing the memory layout for this structure.
