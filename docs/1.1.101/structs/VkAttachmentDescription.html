Parameters object When instantiated with an object, members can be filled directly. Properties VkAttachmentDescription.prototype.flags Number is a bitmask of VkAttachmentDescriptionFlagBits specifying additional properties of the attachment. VkAttachmentDescription.prototype.format Number is a VkFormat value specifying the format of the image view that will be used for the attachment. VkAttachmentDescription.prototype.samples Number is the number of samples of the image as defined in VkSampleCountFlagBits. VkAttachmentDescription.prototype.loadOp Number is a VkAttachmentLoadOp value specifying how the contents of color and depth components of the attachment are treated at the beginning of the subpass where it is first used. VkAttachmentDescription.prototype.storeOp Number is a VkAttachmentStoreOp value specifying how the contents of color and depth components of the attachment are treated at the end of the subpass where it is last used. VkAttachmentDescription.prototype.stencilLoadOp Number is a VkAttachmentLoadOp value specifying how the contents of stencil components of the attachment are treated at the beginning of the subpass where it is first used. VkAttachmentDescription.prototype.stencilStoreOp Number is a VkAttachmentStoreOp value specifying how the contents of stencil components of the attachment are treated at the end of the last subpass where it is used. VkAttachmentDescription.prototype.initialLayout Number is the layout the attachment image subresource will be in when a render pass instance begins. VkAttachmentDescription.prototype.finalLayout Number is the layout the attachment image subresource will be transitioned to when a render pass instance ends. Default Properties VkAttachmentDescription.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkAttachmentDescription.memoryLayout Object Object describing the memory layout for this structure.
