 Categories vkCmdBindDescriptorSets Binds descriptor sets to a command buffer Syntax 
vkCmdBindDescriptorSets();
Parameters commandBuffer VkCommandBuffer is the command buffer that the descriptor sets will be bound to. pipelineBindPoint Number is a elink:VkPipelineBindPoint indicating whether the descriptors will be used by graphics pipelines or compute pipelines. There is a separate set of bind points for each of graphics and compute, so binding one does not disturb the other. layout VkPipelineLayout is a slink:VkPipelineLayout object used to program the bindings. firstSet Number is the set number of the first descriptor set to be bound. descriptorSetCount Number is the number of elements in the pname:pDescriptorSets array. pDescriptorSets Array[VkDescriptorSet] is an array of handles to slink:VkDescriptorSet objects describing the descriptor sets to write to. dynamicOffsetCount Number is the number of dynamic offsets in the pname:pDynamicOffsets array. pDynamicOffsets Uint32Array is a pointer to an array of code:uint32_t values specifying dynamic offsets.
