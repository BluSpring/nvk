 Categories vkGetPhysicalDeviceSurfaceCapabilities2KHR Reports capabilities of a surface on a physical device Syntax 
VkResult vkGetPhysicalDeviceSurfaceCapabilities2KHR();
Parameters physicalDevice VkPhysicalDevice is the physical device that will be associated with the swapchain to be created, as described for flink:vkCreateSwapchainKHR. pSurfaceInfo VkPhysicalDeviceSurfaceInfo2KHR reference to an instance of the slink:VkPhysicalDeviceSurfaceInfo2KHR structure, describing the surface and other fixed parameters that would be consumed by flink:vkCreateSwapchainKHR. pSurfaceCapabilities VkSurfaceCapabilities2KHR reference to an instance of the slink:VkSurfaceCapabilities2KHR structure in which the capabilities are returned.
