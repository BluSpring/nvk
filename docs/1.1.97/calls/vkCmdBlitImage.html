 Categories vkCmdBlitImage Copy regions of an image, potentially performing format conversion, Syntax 
void vkCmdBlitImage();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. srcImage VkImage is the source image. srcImageLayout Number is the layout of the source image subresources for the blit. dstImage VkImage is the destination image. dstImageLayout Number is the layout of the destination image subresources for the blit. regionCount Number is the number of regions to blit. pRegions Array[VkImageBlit] is an array of VkImageBlit structures specifying the regions to blit. filter Number is a VkFilter specifying the filter to apply if the blits require scaling.
