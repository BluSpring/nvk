 Categories vkUpdateDescriptorSetWithTemplate Update the contents of a descriptor set object using an update template Syntax 
void vkUpdateDescriptorSetWithTemplate();
Parameters device VkDevice is the logical device that updates the descriptor sets. descriptorSet VkDescriptorSet is the descriptor set to update descriptorUpdateTemplate VkDescriptorUpdateTemplate is the slink:VkDescriptorUpdateTemplate which specifies the update mapping between pname:pData and the descriptor set to update. pData ArrayBuffer is a pointer to memory which contains one or more structures of slink:VkDescriptorImageInfo, slink:VkDescriptorBufferInfo, or slink:VkBufferView used to write the descriptors.
