Parameters commandBuffer VkCommandBuffer is the command buffer in which the push constant update will be recorded. layout VkPipelineLayout is the pipeline layout used to program the push constant updates. stageFlags Number is a bitmask of elink:VkShaderStageFlagBits specifying the shader stages that will use the push constants in the updated range. offset Number is the start offset of the push constant range to update, in units of bytes. size Number bytes containing the new push constant values. pValues null
