 Categories vkCmdResolveImage Resolve regions of an image Syntax 
void vkCmdResolveImage();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. srcImage VkImage is the source image. srcImageLayout Number is the layout of the source image subresources for the resolve. dstImage VkImage is the destination image. dstImageLayout Number is the layout of the destination image subresources for the resolve. regionCount Number is the number of regions to resolve. pRegions Array[VkImageResolve] is an array of slink:VkImageResolve structures specifying the regions to resolve.
