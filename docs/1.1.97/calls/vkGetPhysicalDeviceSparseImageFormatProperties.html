 Categories vkGetPhysicalDeviceSparseImageFormatProperties Retrieve properties of an image format applied to sparse images Syntax 
void vkGetPhysicalDeviceSparseImageFormatProperties();
Parameters physicalDevice VkPhysicalDevice is the physical device from which to query the sparse image capabilities. format Number is the image format. type Number is the dimensionality of image. samples Number is the number of samples per texel as defined in elink:VkSampleCountFlagBits. usage Number is a bitmask describing the intended usage of the image. tiling Number is the tiling arrangement of the texel blocks in memory. pPropertyCount Object is a pointer to an integer related to the number of sparse format properties available or queried, as described below. pProperties Array[VkSparseImageFormatProperties] is either `NULL` or a pointer to an array of slink:VkSparseImageFormatProperties structures.
