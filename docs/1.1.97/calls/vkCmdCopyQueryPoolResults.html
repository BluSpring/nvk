 Categories vkCmdCopyQueryPoolResults Copy the results of queries in a query pool to a buffer object Syntax 
vkCmdCopyQueryPoolResults();
Parameters commandBuffer VkCommandBuffer is the command buffer into which this command will be recorded. queryPool VkQueryPool is the query pool managing the queries containing the desired results. firstQuery Number is the initial query index. queryCount Number is the number of queries. pname:firstQuery and pname:queryCount together define a range of queries. dstBuffer VkBuffer is a slink:VkBuffer object that will receive the results of the copy command. dstOffset Number stride Number is the stride in bytes between results for individual queries within pname:dstBuffer. The required size of the backing memory for pname:dstBuffer is determined as described above for flink:vkGetQueryPoolResults. flags Number is a bitmask of elink:VkQueryResultFlagBits specifying how and when results are returned.
