VkBindBufferMemoryInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkBindBufferMemoryInfo.prototype.buffer VkBuffer is the buffer to be attached to memory. VkBindBufferMemoryInfo.prototype.memory VkDeviceMemory which is to be bound to the buffer. The number of bytes returned in the VkMemoryRequirements::size member in memory, starting from memoryOffset bytes, will be bound to the specified buffer.
