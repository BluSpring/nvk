VkAttachmentDescription2KHR.prototype.flags Number is a bitmask of elink:VkAttachmentDescriptionFlagBits specifying additional properties of the attachment. VkAttachmentDescription2KHR.prototype.format Number is a elink:VkFormat value specifying the format of the image that will be used for the attachment. VkAttachmentDescription2KHR.prototype.samples Number is the number of samples of the image as defined in elink:VkSampleCountFlagBits. VkAttachmentDescription2KHR.prototype.loadOp Number is a elink:VkAttachmentLoadOp value specifying how the contents of color and depth components of the attachment are treated at the beginning of the subpass where it is first used. VkAttachmentDescription2KHR.prototype.storeOp Number is a elink:VkAttachmentStoreOp value specifying how the contents of color and depth components of the attachment are treated at the end of the subpass where it is last used. VkAttachmentDescription2KHR.prototype.stencilLoadOp Number is a elink:VkAttachmentLoadOp value specifying how the contents of stencil components of the attachment are treated at the beginning of the subpass where it is first used. VkAttachmentDescription2KHR.prototype.stencilStoreOp Number is a elink:VkAttachmentStoreOp value specifying how the contents of stencil components of the attachment are treated at the end of the last subpass where it is used. VkAttachmentDescription2KHR.prototype.initialLayout Number is the layout the attachment image subresource will be in when a render pass instance begins. VkAttachmentDescription2KHR.prototype.finalLayout Number is the layout the attachment image subresource will be transitioned to when a render pass instance ends.
