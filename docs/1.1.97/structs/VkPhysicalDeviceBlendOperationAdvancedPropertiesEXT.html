VkPhysicalDeviceBlendOperationAdvancedPropertiesEXT.prototype.advancedBlendMaxColorAttachments Number is one greater than the highest color attachment index that can: be used in a subpass, for a pipeline that uses an <>. VkPhysicalDeviceBlendOperationAdvancedPropertiesEXT.prototype.advancedBlendIndependentBlend Number specifies whether advanced blend operations can: vary per-attachment. VkPhysicalDeviceBlendOperationAdvancedPropertiesEXT.prototype.advancedBlendNonPremultipliedDstColor Number specifies whether the destination color can: be treated as non-premultiplied.
