Parameters object When instantiated with an object, members can be filled directly. Properties VkAttachmentReference.prototype.attachment Number is the index of the attachment of the render pass, and corresponds to the index of the corresponding element in the pAttachments array of the VkRenderPassCreateInfo structure. If any color or depth/stencil attachments are VK_ATTACHMENT_UNUSED, then no writes occur for those attachments. VkAttachmentReference.prototype.layout Number is a VkImageLayout value specifying the layout the attachment uses during the subpass. Default Properties VkAttachmentReference.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkAttachmentReference.memoryLayout Object Object describing the memory layout for this structure.
