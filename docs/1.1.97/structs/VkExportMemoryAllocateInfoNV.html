Parameters object When instantiated with an object, members can be filled directly. Properties VkExportMemoryAllocateInfoNV.prototype.sType Number is the type of this structure. VkExportMemoryAllocateInfoNV.prototype.pNext null is null or a reference to an extension-specific structure. VkExportMemoryAllocateInfoNV.prototype.handleTypes Number is a bitmask of VkExternalMemoryHandleTypeFlagBitsNV specifying one or more memory handle types that may: be exported. Multiple handle types may: be requested for the same allocation as long as they are compatible, as reported by vkGetPhysicalDeviceExternalImageFormatPropertiesNV. Default Properties VkExportMemoryAllocateInfoNV.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
