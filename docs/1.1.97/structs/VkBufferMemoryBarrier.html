Parameters object When instantiated with an object, members can be filled directly. Properties VkBufferMemoryBarrier.prototype.sType Number is the type of this structure. VkBufferMemoryBarrier.prototype.pNext null is null or a reference to an extension-specific structure. VkBufferMemoryBarrier.prototype.srcAccessMask Number is a bitmask of VkAccessFlagBits specifying a <>. VkBufferMemoryBarrier.prototype.dstAccessMask Number is a bitmask of VkAccessFlagBits specifying a <>. VkBufferMemoryBarrier.prototype.srcQueueFamilyIndex Number is the source queue family for a <>. VkBufferMemoryBarrier.prototype.dstQueueFamilyIndex Number is the destination queue family for a <>. VkBufferMemoryBarrier.prototype.buffer VkBuffer is a handle to the buffer whose backing memory is affected by the barrier. VkBufferMemoryBarrier.prototype.offset Number is an offset in bytes into the backing memory for buffer; this is relative to the base offset as bound to the buffer (see vkBindBufferMemory). VkBufferMemoryBarrier.prototype.size Number is a size in bytes of the affected area of backing memory for buffer, or VK_WHOLE_SIZE to use the range from offset to the end of the buffer. Default Properties VkBufferMemoryBarrier.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
