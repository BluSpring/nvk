Properties VkPhysicalDeviceSubgroupProperties.prototype.sType Number is the type of this structure. VkPhysicalDeviceSubgroupProperties.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceSubgroupProperties.prototype.subgroupSize Number is the number of invocations in each subgroup. This will match any <SubgroupSize>> decorated variable used in any shader module created on this device. subgroupSize is at least 1 if any of the physical device's queues support VK_QUEUE_GRAPHICS_BIT or VK_QUEUE_COMPUTE_BIT. VkPhysicalDeviceSubgroupProperties.prototype.supportedStages Number is a bitfield of VkShaderStageFlagBits describing the shader stages that subgroup operations are supported in. supportedStages will have the VK_SHADER_STAGE_COMPUTE_BIT bit set if any of the physical device's queues support VK_QUEUE_COMPUTE_BIT. VkPhysicalDeviceSubgroupProperties.prototype.supportedOperations Number is a bitmask of VkSubgroupFeatureFlagBits specifying the sets of subgroup operations supported on this device. supportedOperations will have the VK_SUBGROUP_FEATURE_BASIC_BIT bit set if any of the physical device's queues support VK_QUEUE_GRAPHICS_BIT or VK_QUEUE_COMPUTE_BIT. VkPhysicalDeviceSubgroupProperties.prototype.quadOperationsInAllStages Boolean is a boolean that specifies whether <> are available in all stages, or are restricted to fragment and compute stages. Default Properties VkPhysicalDeviceSubgroupProperties.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
