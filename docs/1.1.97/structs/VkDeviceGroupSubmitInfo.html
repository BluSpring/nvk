Parameters object When instantiated with an object, members can be filled directly. Properties VkDeviceGroupSubmitInfo.prototype.sType Number is the type of this structure. VkDeviceGroupSubmitInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkDeviceGroupSubmitInfo.prototype.waitSemaphoreCount Number is the number of elements in the pWaitSemaphoreDeviceIndices array. VkDeviceGroupSubmitInfo.prototype.pWaitSemaphoreDeviceIndices Uint32Array is an array of device indices indicating which physical device executes the semaphore wait operation in the corresponding element of VkSubmitInfo::pWaitSemaphores. VkDeviceGroupSubmitInfo.prototype.commandBufferCount Number is the number of elements in the pCommandBufferDeviceMasks array. VkDeviceGroupSubmitInfo.prototype.pCommandBufferDeviceMasks Uint32Array is an array of device masks indicating which physical devices execute the command buffer in the corresponding element of VkSubmitInfo::pCommandBuffers. A physical device executes the command buffer if the corresponding bit is set in the mask. VkDeviceGroupSubmitInfo.prototype.signalSemaphoreCount Number is the number of elements in the pSignalSemaphoreDeviceIndices array. VkDeviceGroupSubmitInfo.prototype.pSignalSemaphoreDeviceIndices Uint32Array is an array of device indices indicating which physical device executes the semaphore signal operation in the corresponding element of VkSubmitInfo::pSignalSemaphores. Default Properties VkDeviceGroupSubmitInfo.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
