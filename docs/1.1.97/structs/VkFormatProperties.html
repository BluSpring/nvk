Parameters object When instantiated with an object, members can be filled directly. Properties VkFormatProperties.prototype.linearTilingFeatures Number is a bitmask of VkFormatFeatureFlagBits specifying features supported by images created with a tiling parameter of VK_IMAGE_TILING_LINEAR. VkFormatProperties.prototype.optimalTilingFeatures Number is a bitmask of VkFormatFeatureFlagBits specifying features supported by images created with a tiling parameter of VK_IMAGE_TILING_OPTIMAL. VkFormatProperties.prototype.bufferFeatures Number is a bitmask of VkFormatFeatureFlagBits specifying features supported by buffers. Default Properties VkFormatProperties.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkFormatProperties.memoryLayout Object Object describing the memory layout for this structure.
