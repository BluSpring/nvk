Parameters object When instantiated with an object, members can be filled directly. Properties VkSparseMemoryBind.prototype.resourceOffset Number is the offset into the resource. VkSparseMemoryBind.prototype.size Number is the size of the memory region to be bound. VkSparseMemoryBind.prototype.memory VkDeviceMemory is the slink:VkDeviceMemory object that the range of the resource is bound to. If pname:memory is dlink:VK_NULL_HANDLE, the range is unbound. VkSparseMemoryBind.prototype.memoryOffset Number is the offset into the slink:VkDeviceMemory object to bind the resource range to. If pname:memory is dlink:VK_NULL_HANDLE, this value is ignored. VkSparseMemoryBind.prototype.flags Number is a bitmask of elink:VkSparseMemoryBindFlagBits specifying usage of the binding operation. Default Properties VkSparseMemoryBind.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkSparseMemoryBind.prototype.memoryLayout Object Object describing the memory layout for this structure's members.
