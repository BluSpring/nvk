Parameters object When instantiated with an object, members can be filled directly. Properties VkImageViewCreateInfo.prototype.sType Number is the type of this structure. VkImageViewCreateInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkImageViewCreateInfo.prototype.flags Number is a bitmask of VkImageViewCreateFlagBits describing additional parameters of the image view. VkImageViewCreateInfo.prototype.image VkImage is a VkImage on which the view will be created. VkImageViewCreateInfo.prototype.viewType Number is a VkImageViewType value specifying the type of the image view. VkImageViewCreateInfo.prototype.format Number is a VkFormat describing the format and type used to interpret texel blocks in the image. VkImageViewCreateInfo.prototype.components VkComponentMapping is a VkComponentMapping specifies a remapping of color components (or of depth or stencil components after they have been converted into color components). VkImageViewCreateInfo.prototype.subresourceRange VkImageSubresourceRange is a VkImageSubresourceRange selecting the set of mipmap levels and array layers to be accessible to the view. Default Properties VkImageViewCreateInfo.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
