Parameters object When instantiated with an object, members can be filled directly. Properties VkAccelerationStructureMemoryRequirementsInfoNV.prototype.sType Number is the type of this structure. VkAccelerationStructureMemoryRequirementsInfoNV.prototype.pNext null is null or a reference to an extension-specific structure. VkAccelerationStructureMemoryRequirementsInfoNV.prototype.type Number selects the type of memory requirement being queried. {#0#} returns the memory requirements for the object itself. ename:VK_ACCELERATION_STRUCTURE_MEMORY_REQUIREMENTS_TYPE_BUILD_SCRATCH_NV returns the memory requirements for the scratch memory when doing a build. {#1#} returns the memory requirements for the scratch memory when doing an update.
