Parameters object When instantiated with an object, members can be filled directly. Properties VkAttachmentReference2KHR.prototype.sType Number is the type of this structure. VkAttachmentReference2KHR.prototype.pNext null is null or a reference to an extension-specific structure. VkAttachmentReference2KHR.prototype.attachment Number is either a number value identifying an attachment at the corresponding index in VkRenderPassCreateInfo::pAttachments, or VK_ATTACHMENT_UNUSED to signify that this attachment is not used. VkAttachmentReference2KHR.prototype.layout Number is a VkImageLayout value specifying the layout the attachment uses during the subpass. VkAttachmentReference2KHR.prototype.aspectMask Number is a mask of which aspect(s) can be accessed within the specified subpass as an input attachment.
