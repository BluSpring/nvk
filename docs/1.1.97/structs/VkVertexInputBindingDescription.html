Parameters object When instantiated with an object, members can be filled directly. Properties VkVertexInputBindingDescription.prototype.binding Number is the binding number that this structure describes. VkVertexInputBindingDescription.prototype.stride Number is the distance in bytes between two consecutive elements within the buffer. VkVertexInputBindingDescription.prototype.inputRate Number is a VkVertexInputRate value specifying whether vertex attribute addressing is a function of the vertex index or of the instance index. Default Properties VkVertexInputBindingDescription.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
