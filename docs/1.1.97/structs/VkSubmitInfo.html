Parameters object When instantiated with an object, members can be filled directly. Properties VkSubmitInfo.prototype.sType Number is the type of this structure. VkSubmitInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkSubmitInfo.prototype.waitSemaphoreCount Number is the number of semaphores upon which to wait before executing the command buffers for the batch. VkSubmitInfo.prototype.pWaitSemaphores Array[VkSemaphore] is an array of semaphores upon which to wait before the command buffers for this batch begin execution. If semaphores to wait on are provided, they define a <>. VkSubmitInfo.prototype.pWaitDstStageMask Int32Array is an array of pipeline stages at which each corresponding semaphore wait will occur. VkSubmitInfo.prototype.commandBufferCount Number is the number of command buffers to execute in the batch. VkSubmitInfo.prototype.pCommandBuffers Array[VkCommandBuffer] is an array of command buffers to execute in the batch. VkSubmitInfo.prototype.signalSemaphoreCount Number is the number of semaphores to be signaled once the commands specified in pCommandBuffers have completed execution. VkSubmitInfo.prototype.pSignalSemaphores Array[VkSemaphore] is an array of semaphores which will be signaled when the command buffers for this batch have completed execution. If semaphores to be signaled are provided, they define a <>. Default Properties VkSubmitInfo.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkSubmitInfo.prototype.memoryLayout Object Object describing the memory layout for this structure's members.
