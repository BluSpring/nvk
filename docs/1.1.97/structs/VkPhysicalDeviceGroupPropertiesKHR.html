Parameters object When instantiated with an object, members can be filled directly. Properties VkPhysicalDeviceGroupPropertiesKHR.prototype.sType Number is the type of this structure. VkPhysicalDeviceGroupPropertiesKHR.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceGroupPropertiesKHR.prototype.physicalDeviceCount Number is the number of physical devices in the group. VkPhysicalDeviceGroupPropertiesKHR.prototype.physicalDevices Array[Number] is an array of physical device handles representing all physical devices in the group. The first physicalDeviceCount elements of the array will be valid. VkPhysicalDeviceGroupPropertiesKHR.prototype.subsetAllocation Number specifies whether logical devices created from the group support allocating device memory on a subset of devices, via the deviceMask member of the VkMemoryAllocateFlagsInfo. If this is VK_FALSE, then all device memory allocations are made across all physical devices in the group. If physicalDeviceCount is `1`, then pname:subsetAllocation must: be VK_FALSE. Default Properties VkPhysicalDeviceGroupPropertiesKHR.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
