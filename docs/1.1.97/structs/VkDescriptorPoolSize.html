Parameters object When instantiated with an object, members can be filled directly. Properties VkDescriptorPoolSize.prototype.type Number is the type of descriptor. VkDescriptorPoolSize.prototype.descriptorCount Number is the number of descriptors of that type to allocate. If pname:type is ename:VK_DESCRIPTOR_TYPE_INLINE_UNIFORM_BLOCK_EXT then pname:descriptorCount is the number of bytes to allocate for descriptors of this type. Default Properties VkDescriptorPoolSize.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkDescriptorPoolSize.prototype.memoryLayout Object Object describing the memory layout for this structure's members.
