Properties VkLayerProperties.prototype.layerName String is a string specifying the name of the layer. Use this name in the ppEnabledLayerNames array passed in the VkInstanceCreateInfo structure to enable this layer for an instance. VkLayerProperties.prototype.specVersion Number is the Vulkan version the layer was written to, encoded as described in the <> section. VkLayerProperties.prototype.implementationVersion Number is the version of this layer. It is a number, increasing with backward compatible changes. VkLayerProperties.prototype.description String is a string providing additional details that can be used by the application to identify the layer. Default Properties VkLayerProperties.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkLayerProperties.memoryLayout Object Object describing the memory layout for this structure.
