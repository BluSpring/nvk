Parameters object When instantiated with an object, members can be filled directly. Properties VkQueryPoolCreateInfo.prototype.sType Number is the type of this structure. VkQueryPoolCreateInfo.prototype.pNext null is `NULL` or a pointer to an extension-specific structure. VkQueryPoolCreateInfo.prototype.flags Number is reserved for future use. VkQueryPoolCreateInfo.prototype.queryType Number is a elink:VkQueryType value specifying the type of queries managed by the pool. VkQueryPoolCreateInfo.prototype.queryCount Number is the number of queries managed by the pool. VkQueryPoolCreateInfo.prototype.pipelineStatistics Number is a bitmask of elink:VkQueryPipelineStatisticFlagBits specifying which counters will be returned in queries on the new pool, as described below in <>. Default Properties VkQueryPoolCreateInfo.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
