VkPipelineCacheCreateInfo.prototype.initialDataSize Number VkPipelineCacheCreateInfo.prototype.pInitialData ArrayBuffer is a reference to previously retrieved pipeline cache data. If the pipeline cache data is incompatible (as defined below) with the device, the pipeline cache will be initially empty. If initialDataSize is zero, pInitialData is ignored.
